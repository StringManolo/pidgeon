Encrypted Inbox

#### Install
```bash
git clone https://github.com/StringManolo/pidgeon;
cd pidgeon;
```

#### Run Server
```bash
node ./dist/server.js;
```

#### Run Client
```bash
node ./dist/client.js;
```
