#### Install 
```bash
git clone https://github.com/StringManolo/pidgeon;
cd pidgeon;
npm install;
```

#### Run Server
```bash
node ./dist/server.js;
```

#### Run Terminal Client (commands)
```bash
node ./dist/client-cli.js;
```
